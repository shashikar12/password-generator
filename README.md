# Classic Password Generator

A polished password generator built with React, Vite, Framer Motion, Lucide React, and React Hot Toast.

## Preview

![Classic Password Generator UI](public/password-generator-preview.png)

## Features

- Smooth animated layout with a modern classic visual style
- Live password strength feedback
- Copy-to-clipboard action with toast confirmation
- Toggle password visibility
- Generate passwords with numbers and symbols

## Tech Stack

- React 19
- Vite
- Framer Motion
- Tailwind CSS
- Lucide React
- React Hot Toast

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
